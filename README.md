# nlp-transformer-implementation

This project is a minimal implementation of the Transformer architecture based on the paper  
**"Attention Is All You Need" (Vaswani et al., 2017)**.

## Overview

The goal of this project is to understand how Transformers work by implementing them from scratch, without relying on high-level libraries.


## Motivation

Transformers are the foundation of modern NLP and large language models.  
This project is built for educational purposes to deeply understand:

- How attention works
- Why Transformers outperform RNNs
- Internal architecture of encoder-decoder models

## Requirements

- Python 3.8+
- PyTorch (recommended)

Install dependencies:

```bash
pip install torch numpy
```
